Synthetic Data

The Synthetic Data Vault (SDV) on Google Colab demonstrates how synthetic data generation can replicate real datasets, enabling secure and privacy-preserving analysis. It supports various data types, ensuring realistic, high-quality synthetic data for testing, training, and analytics without compromising sensitive information.
Tech stack: Google Colab, Synthetic Data Vault (SDV)

Data Visualization

Apache Superset is an open-source data visualization tool for creating interactive dashboards and exploring datasets. It supports multiple databases, offers a user-friendly interface, and enables seamless chart creation without coding. Ideal for data professionals, it simplifies analytics and enhances data-driven decision-making.
